CherryDo
======

CherryPy web generators

Examples
--------

Create new project

```
cherrydo new <project_name>
```

Generate controller

```
cherrydo generate controller <controller_name> (<views>)
```

Generate model

```
cherrydo generate model <model_name> (<fields>)
```

DB migrate

```
cherrydo db migrate
```

DB rollback

```
cherrydo db rollback
```
